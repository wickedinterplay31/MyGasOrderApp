# MyGasOrderApp

This project is a simple gas-ordering application built with Flask and SQLite.

## Features
- Buyer and seller login sections
- Sign up page with optional email and phone
- Buyer can order 6kg, 13kg, 3kg cylinders or request a refill
- Seller has administrative privileges and can view all orders
- Order report page shows gas items ordered
- SQLite database stored in `gas.db`

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the app:
   ```bash
   python app.py
   ```
3. Open your browser to:
   ```
   http://127.0.0.1:5000/
   ```

## Default seller admin
- Username: `seller`
- Password: `admin123`

## Notes
- The database file `gas.db` is created automatically when the app first runs.
- You can create new buyer or seller accounts via the sign-up page.
